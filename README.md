# Simple Financial Tracking App

A simple web-based application for recording daily income and expenses.

Live Demo: https://alifiogalangtech-rgb.github.io/simple-finance-tracker/

## Description
This application helps individuals or small groups track their financial activities by recording income and expenses and automatically calculating totals and balance.

It is designed to be simple, lightweight, and easy to use without complex features.

## Features
- Add income and expense records  
- View transaction list  
- Automatic calculation of total income, total expenses, and balance  
- Data stored locally in the browser (localStorage)  
- No login and no database required  

## How to Use
1. Open the application in a web browser  
2. Enter the date, description, and amount  
3. Select transaction type (income or expense)  
4. Data will be saved automatically and displayed in the list  

## Notes
This application is intended for simple use cases such as personal finance tracking or small communities.  
It is not designed to replace full accounting systems.

This project is built as a learning and portfolio demonstration.
